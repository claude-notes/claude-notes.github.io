GPT5.1来了！不光更聪明，还会“共情”：新增6种人格风格（附如何使用到GPT5.1教程）

就在今天凌晨，OpenAI正式发布了他们的最新模型 GPT-5.1，对GPT-5系列进行了全面升级。

这次，没有发布会、也没有倒计时，Sam Altman 也只是在清晨的 X 上发了一句话 ——“GPT-5.1 出来了！这是一次不错的升级。” 就在这不经意的一句话里，OpenAI 宣告了大模型的新阶段：从单纯追求算力与推理能力，进入“智商 + 情商”双线并行的时代。

本次更新包含两个核心模型，未来还会加入更强的 Pro 版。

GPT5.1核心模型更新要点

GPT-5.1 目前采用双模型架构，并新增 “GPT-5.1 Auto” 自动调度系统，可根据任务自动选择最佳模型。

GPT-5.1 Instant： 首次引入“自适应推理”

作为高频使用的默认模型，Instant 的核心升级是 自适应推理（Adaptive Reasoning），主打一个”快准稳”：

  • 简单任务（天气、闲聊等）→ 毫秒级响应
  • 复杂任务（数学、代码、逻辑推理）→ 自动延长思考时间，调用更深的推理链

官方数据显示，Instant 在 AIME 2025 数学竞赛、Codeforces 编程评测等复杂任务中，准确率显著上升。

同时，它的语言风格全面升级：

  • 更温暖、更自然
  • 回应更贴合情境
  • 指令遵循度更高
  • 偶尔还能“抖个机灵”

模型终于摆脱“机械罗列式回答”，更像一个真正懂你的私人教练、生活顾问或工作伙伴。

GPT-5.1 Thinking：深度推理的“智能调控”

高级推理模型，更像一位“慢工出细活”的专家型模型，它的升级重点在于 思考深度的动态调节，主打一个”深透强”：

  • 简单任务：响应速度甚至比旧版更快
  • 复杂任务：自动增配计算资源、增加推理步数

同时，它的表达方式实现了重大改善：

  • 尽量减少专业术语
  • 更注重可读性
  • 解释技术类问题时更像老师，而不是教科书

在高情绪场景下，表现也更加自然与稳重。官方示例中，当用户因会议前把咖啡打翻而焦虑时，GPT-5.1 Thinking 不再一味给建议，而是先共情、再引导情绪转变，显得更像“懂人话的 AI 同伴”。

六大官方人格 + 深度自定义

GPT-5.1 在个性化交互上迈出关键一步。

此次更新还包括了语调控制功能，新增了包括“默认”、“友好”（原名“倾听者”）、“高效”（原名“机器人”）、“专业”、“坦率”和“古怪”在内的优化预设，同时保留了“愤世嫉俗”（原名“愤世嫉俗”）和“书呆子”（原名“书呆子”）两种预设。

大家可以在官方 账号设置 -> 个性化 -> 基本风格和语调 中进行相对应调整

用户还可手动调节：

  • 语言简洁度
  • 热情程度
  • 解释深度
  • 表情符号使用频率

而且 AI 会主动学习你的偏好。

如果你频繁喜欢它的调侃式回应，它会贴心地问：

“需要我以后一直这样和你聊天吗？”

点击确认，它就能形成稳定人格，避免“人设崩塌”。

哪些用户可以使用到？

👉 老样子，GPT-5.1 将首先面向付费用户（Pro、Plus、Go、Business）推出，随后面向免费用户和未登录用户推出。

原 GPT-5 模型将在“旧版模型”下拉菜单中保留三个月。

本周晚些时候将推出 API 访问权限，届时将提供 gpt-5.1-chat-latest（即时版）和 gpt-5.1（思考版），两者均具备自适应推理功能。

GPT-5.1 Pro 将很快取代 GPT-5 Pro。

另外AI编程助手 Codex 这边，目前还未上新 GPT-5.1，相信也会在不久之后进行更新。
