OpenClaw + GPT-5.4，手把手教你无需购买 API 也能上手使用【附带GPT升级教程】

就在昨天，热门智能体项目 OpenClaw 现已更新至 2026.3.7 版本，带来了上下文引擎插件、lossless-claw 等重磅功能，并进一步扩展了对一系列新模型的支持。

而本次更新中，我们可以发现其中对新模型的支持中就包含了OpenAI最新模型GPT-5.4，这个天选模型发布之后，大家也是争先恐后地想要在OpenClaw接入该模型，但奈何暂不支持授权切换。

相信很多人手中都有自己的gpt账号，现在不管是免费、Plus、Business、Pro 订阅的用户，都可以有额度去使用到ChatGPT附带的Codex。

早在二月份的时候，OpenAI首席执行官就宣布Free/Go用户目前也有 Codex 权限，只不过在额度上可能会与其他付费用户有一定的差距

如果你的GPT经常升级失败，也没有海外信用卡、虚拟卡的话，可以可以看一下往期文章👇

升级OpenClaw版本

首先需要升级OpenClaw的版本，如果你的版本不是2026.3.7版本的话，是不能够授权使用到GPT-5.4的~

需要在服务器下运行升级命令

升级完成之后可以查看一下当前版本

运行 onboard 向导

在执行之前可以先备份一份防止出错，也便于我们回溯

然后运行向导，进行授权openai的codex

之后向导会提出几个问题让你进行选择，可以根据图片中的依次选择Yes、QuickStart、Use existing values

👉注意最后一个不要选择Reset！！不然你之前对小龙虾的一些设置就会全没了

上面配置选择之后就会看到OpenAI Codex OAuth，下面会贴出账号授权url

将这个 URL 复制到本地电脑的浏览器里打开，用 ChatGPT 账号登录，点击授权

（如果是本地电脑跑 OpenClaw 的，浏览器弹出来，登录授权即可）

之后会跳转到一个本地地址，类似http://localhost:1455/auth/callback?code=

正常是打不开的，显示如下

我们将这个地址复制到openclaw刚才的配置下，完成授权操作

跳过后续配置

后续的配置一律跳过即可

频道选择那里选 Skip for now

Configure skills now? (recommended)选择No

Enable hooks?选择 skip for now

Gateway service already installed 选择 Restart

之后的How do you want to hatch your bot?选择Do this later

至此，龙虾就配置好了。

设置GPT-5.4模型

运行设置模型命令

运行后可以发现默认模型已切换为gpt-5.4

如何查看额度？

点击gpt对话页面左上角的codex，进入codex页面

找到右上角的Codex设置，查看使用情况即可查看codex额度

Codex额度与ChatGPT的对话是相互隔开，互不影响的。

如果你没有代码需要的话，可以直接将Codex用在OpenClaw中，付费用户（Plus、Business、Pro）用户正常额度都是够用的。

在御三家中，只有OpenAI是支持授权登录OpenClaw等第三方工具的，其他家像Anthropic、Google是发出声明不允许授权的，可能会有封号的风险，谨慎使用！
