Claude Code 新功能上线，正式开启全民赛博养宠模式！！！

就在前两天 Claude Code意外泄露源码，当时人们发现里面有一个虚拟宠物伙伴Buddy模式尚未发布，而现在，它来了。

什么是 Buddy 模式？

在最新版本的 Claude Codev2.1.89 中，你的编辑器旁边会出现一只可爱的小宠物，它会在你编程时陪伴你，偶尔对你的代码发表评论，甚至在你遇到困难时给出建议。它会以完整的电子宠物风格出现在你的 Claude 代码终端里，让编程过程变得更加有趣和互动。

可以通过 npm i -g @anthropic-ai/claude-code自行更新最新版本v2.1.89的

重新进入 claude 输入 /buddy 即可查看自己的宠物属性啦，每只宠物都有自己的名字

宠物的五大属性系统

每只宠物都有独特的属性值，这些属性会影响它们的行为和互动方式：

1. DEBUGGING（调试能力）

  • 影响宠物发现代码问题的敏锐度
  • 数值越高，越能帮你找到隐藏的 bug
  • 提供调试建议的频率和质量也会提升

2. PATIENCE（耐心值）

  • 决定宠物在长时间任务中的表现
  • 高耐心值的宠物不会催促你
  • 对重复性问题的容忍度更高

3. CHAOS（混乱度）

  • 影响宠物的随机性和创意性
  • 高混乱度可能带来意想不到的建议
  • 让编程过程充满惊喜（或惊吓？）

4. WISDOM（智慧值）

  • 决定建议的深度和洞察力
  • 高智慧值的宠物能给出更专业的指导
  • 像一位经验丰富的导师

5. SNARK（讽刺/吐槽值）

  • 影响评论的幽默感和讽刺程度
  • 数值越低越友善温和
  • 数值越高越毒舌（但也更有趣）

宠物类型与稀有度

目前已知的宠物系统包含：

稀有度分级

  • COMMON（普通）：最常见的宠物
  • UNCOMMON（不寻常）：不寻常的宠物
  • RARE（稀有）：可能需要特定条件解锁
  • EPIC（史诗）：更加稀有的宠物类型
  • LEGENDARY（传说）：最稀有的宠物

可能还有其他稀有度或类型等待解锁。

宠物种类

根据目前的信息和社区反馈，可能包含以下类型：

鸭（duck）、鹅（goose）、斑点（blob）、猫（cat）、龙（dragon）、章鱼（octopus）、猫头鹰（owl）、企鹅（penguin）、乌龟（turtle）、蜗牛（snail）、鬼（ghost）、蝾螈（axolotl）、水豚（capybara）、仙人掌（cactus）、机器人（robot）、兔子（rabbit）、蘑菇（mushroom）、大胖墩（chonk）．．．

目前已知有18种（像极了小时候玩的宠物游戏了哈哈哈）

五大好玩玩法

1. 直接对话互动

在输入/buddy之后，宠物会出现在你的输入框右边，你可以在消息中直接呼唤宠物的名字：

它会在气泡中回应你（注意：是宠物回应，不是 Claude 本体）

2. 属性养成系统

宠物的五大属性可能会随着使用而变化：

  • 经常调试代码 → DEBUGGING 值提升
  • 耐心处理复杂任务 → PATIENCE 值增长
  • 尝试创新方案 → CHAOS 值波动
  • 深度学习新知识 → WISDOM 值提高

就像真正的电子宠物养成游戏！

3. 观察评论与反馈

宠物会根据自己的属性对你的操作发表评论：

  • 高 WISDOM 的宠物：给出深刻的架构建议
  • 高 SNARK 的宠物：吐槽你的代码风格
  • 高 DEBUGGING 的宠物：快速指出潜在问题
  • 高 CHAOS 的宠物：提出脑洞大开的解决方案

4. 探索稀有宠物

通过不同的编程行为和成就，可能解锁更多稀有宠物：

  • 完成特定数量的 commit
  • 修复一定数量的 bug
  • 连续编程时长达到里程碑
  • 使用特定编程语言达到熟练度

5. 个性化编程体验

不同属性组合的宠物，会带来完全不同的体验：

  • 高 WISDOM + 低 SNARK：温和的导师型
  • 高 DEBUGGING + 高 SNARK：毒舌但有用的代码审查员
  • 高 CHAOS + 高 PATIENCE：创意十足但不急躁的伙伴
  • 全属性均衡：全能型编程助手
